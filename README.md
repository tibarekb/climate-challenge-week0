## About The Project


This project is about conducting an exploratory analysis of historical climate data across five African nations. The analysis surfaces key climate trends, seasonal patterns, and anomalies that will inform data-driven position. It aims at clear, evidence-backed insights that highlight Africa's unique climate vulnerabilities and support the continent’s climate policy narrative.

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* Python
  
### Installation

Follow the steps below 

1. Clone the repo
   ```sh
   git clone https://github.com/tibarekb/climate-challenge-week0.git
   ```
2. Install  packages
   ```sh
   pip install -r requirements.txt
   ```
3. Change git remote url to avoid accidental pushes to base project
   ```sh
   git remote set-url origin github_username/repo_name
   git remote -v # confirm the changes
   ```
